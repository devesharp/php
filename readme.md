# Devesharp PHP

### Console

Comandos para criar classes rápidas:

```shell script
# Criar todos as classes
php artisan ds:all 

# Criar serviço
php artisan ds:service

# Criar repositório
php artisan ds:repository

# Criar validator
php artisan ds:validator

# Criar Policy
php artisan ds:policy

# Criar transfomer
php artisan ds:transfomer 
```
